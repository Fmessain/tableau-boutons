# tableau-boutons

Extension Tableau affichant les paramètres du dashboard sous forme de boutons interactifs.

## Utilisation

1. Héberger les fichiers sur un serveur web (ou `localhost`)
2. Dans Tableau Desktop/Server, ajouter une extension et pointer vers `index.html`
3. Les paramètres du dashboard s'affichent dans la console

## Fichiers

- `index.html` — point d'entrée de l'extension
- `app.js` — logique principale (récupération des paramètres Tableau)
- `tableau-boutons.trex` — manifeste de l'extension Tableau
